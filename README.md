# NovaTask

A modern, futuristic todo list application built with Next.js, featuring a sleek glass morphism UI, advanced task management, and smooth animations.

## âœ¨ Features

- **Advanced Task Management**: Create, edit, delete, and mark tasks complete
- **Smart Categorization**: Organize tasks by custom categories
- **Priority System**: High, medium, and low priority levels with color coding
- **Due Dates**: Set and track task deadlines
- **Real-time Filtering**: Filter by status (all/active/completed) and category
- **Futuristic UI**: Glass morphism design with smooth animations
- **Responsive Design**: Works perfectly on desktop and mobile
- **Local Storage**: Automatic data persistence
